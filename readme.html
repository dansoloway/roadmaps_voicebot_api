Roadmaps.Fun API Documentation

Welcome to the Roadmaps.Fun API. This API allows you to create AI-generated voices and generate narrations using cloned voices.

1️⃣ Create a Voice Model (POST /clone-voice/)

Endpoint

POST https://api.roadmaps.fun/clone-voice/

Description

Create a voice model by uploading an audio sample of the target voice. Use the returned model_id to generate narrations.

Request Parameters

Parameter Type Required Description
title String ✅ Yes A title for the voice model.
description String ✅ Yes A short description of the cloned voice.
voices File ✅ Yes An audio file (MP3/WAV) containing the reference voice.

Example Request (CURL)

curl -X 'POST' \
  'https://api.roadmaps.fun/clone-voice/' \
  -H 'accept: application/json' \
  -H 'Content-Type: multipart/form-data' \
  -F 'title=test' \
  -F 'description=test' \
  -F 'voices=@/Users/daniel-new2/Downloads/sample_daniel.mp3'

Successful Response

{
  "model_id": "4bcffef7da34455a857bf5a16923b391",
  "message": "Voice model created successfully."
}

Error Responses

HTTP Code Message
400 Missing required fields or incorrect file format.
500 Server error during voice processing.

2️⃣ Generate Narration (POST /generate-narration/)

Endpoint

POST https://api.roadmaps.fun/generate-narration/

Description

Generate an AI narration using a previously created voice model.

Request Parameters

Parameter Type Required Description
model_id String ✅ Yes The ID of the cloned voice model.
text String ✅ Yes The text you want to convert into speech.

Example Request (CURL)

curl -X 'POST' \
  'https://api.roadmaps.fun/generate-narration/' \
  -H 'accept: application/json' \
  -H 'Content-Type: multipart/form-data' \
  -F 'model_id=4bcffef7da34455a857bf5a16923b391' \
  -F 'text=Four score and seven years ago our fathers brought forth, upon this continent, a new nation, conceived in Liberty, and dedicated to the proposition that all men are created equal.' \
  --output narration.mp3

Successful Response

The API returns an MP3 file (narration.mp3) containing the generated narration.

Error Responses

HTTP Code Message
400 Missing required fields or incorrect model_id.
404 Voice model not found.
500 Server error during narration generation.

📌 Notes

  • Ensure the voices file is clear and high-quality for better voice cloning accuracy.
  • If you receive "Voice model not found", verify the model_id returned from /clone-voice/.
  • Use --output filename.mp3 in curl to save the generated audio file.
